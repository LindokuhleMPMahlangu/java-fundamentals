# Java Fundamentals Homework Project

This repository contains homework exercises for Java fundamentals. Each exercise is designed to reinforce concepts and improve programming skills.

## Student Instructions
1. **Clone the repository** to your local machine.
2. Make sure you have [Maven](https://maven.apache.org/) installed.
3. Follow the exercises in the `exercises` folder, completing each module as instructed.
4. Write your unit tests in the corresponding `tests` folder.
5. Submit your completed exercises as instructed below:
   - Create a new branch for your exercises.
   - Push your changes to your GitHub repository.
   - Submit a pull request for review.

## Submission Workflow
- Create a new branch from `main` for your exercise.
- After completing the exercise and writing tests, push your changes.
- Submit a pull request to the `main` branch.
